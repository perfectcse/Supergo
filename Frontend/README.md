# 🎓 CertifyPro Frontend

A modern and responsive certificate generation platform built with **React.js**. Users can create professional certificates, preview them in real time, download them as PDFs, and verify certificates using a unique Certificate ID.

## ✨ Features

- 🎨 Modern and responsive UI
- 📝 Create professional certificates
- 👀 Live certificate preview
- 🎯 Multiple certificate templates
- 📄 Download certificate as PDF
- 🔍 Verify certificates by Certificate ID
- ⚡ Fast React + Vite application
- 📱 Mobile-friendly design

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/perfectcse/certifypro-frontend-.git
```

Move into the project:

```bash
cd certifypro-frontend-
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 🌐 Backend API

The frontend communicates with the CertifyPro Backend REST API built using:

- Node.js
- Express.js
