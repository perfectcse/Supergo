# 🎓 CertifyPro

A full-stack MERN application for creating, verifying, and managing digital certificates.

## 🚀 Tech Stack

- React.js (Vite)
- Node.js
- Express.js
- MongoDB Atlas
- Render

## ✨ Features

- Create digital certificates
- Verify certificates using Certificate ID
- MongoDB Atlas integration
- Responsive UI
- REST API
